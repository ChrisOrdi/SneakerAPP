# SneakerStore Application

## Description
SneakerStore is a modern web application built using Spring Boot and Java, designed for sneaker enthusiasts. This application allows users to browse, search, and purchase the latest and most exclusive sneakers. Featuring a user-friendly interface and a robust backend, SneakerStore ensures a seamless shopping experience.

## Features
- **Browse Sneakers**: View a wide range of sneaker collections.
- **Search Functionality**: Find your desired sneakers with easy-to-use search features.
- **User Authentication**: Secure user registration and login functionality.
- **Shopping Cart**: Add sneakers to your cart for a convenient purchase process.
- **Order History**: Users can view their past orders and track current orders.
- **Responsive Design**: Optimized for various devices and screen sizes.

## Getting Started

### Prerequisites
- Java JDK 8 or later
- Maven 3.2+
- An IDE like IntelliJ IDEA or Eclipse

### Installation
1. **Clone the repository**
   ```
   git clone https://github.com/yourusername/sneakerstore.git
   ```
2. **Navigate to the project directory**
   ```
   cd sneakerstore
   ```
3. **Build the project using Maven**
   ```
   mvn clean install
   ```
4. **Run the application**
   ```
   mvn spring-boot:run
   ```
   The application will start running at `http://localhost:8080`.

### Usage
After starting the application, navigate to `http://localhost:8080` on your browser to start using the SneakerStore application.

## Built With
- **Spring Boot** - The web framework used
- **Maven** - Dependency Management
- **Thymeleaf** - Template engine for the views
- **Spring Security** - For authentication and authorization
- **H2 Database** - Embedded database for development
- **JPA / Hibernate** - ORM for database interaction
